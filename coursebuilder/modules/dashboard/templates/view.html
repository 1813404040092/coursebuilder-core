How to add course carousel

1. Click the 'code' icon at the very top

2. Click where you wish to insert the carousel and paste the code below there:

          <div class="owl-carousel owl-theme course-carousel">
           
            <div class="item">
              content1
            </div>
            
            <div class="item">
              content2
            </div>
            
            <div class="item">
              content3
            </div>
            
          </div>

3. Replace 'content1', 'content2' ... with the content to be displayed to the user

4. You can delete or add additional 'items' as desired. Simply delete or duplicate:

          <div class="item">
            content
          </div>
